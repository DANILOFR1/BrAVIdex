Ícone para o BrAVIdex

Este é um ícone simples que representa o BrAVIdex, um jogo de cartas sobre aves brasileiras.

Como salvar e usar o ícone:

  1. Clique com o botão direito no ícone acima
  2. Selecione "Salvar imagem como..."
  3. Salve como "icon-512x512.png" na pasta "public/images/"
  4. Repita o processo e salve como "icon-192x192.png"
  5. Certifique-se de que o manifest.json aponta para esses arquivos
